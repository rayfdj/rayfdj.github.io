## Code Blocks

Here's an example of syntax-highlighted code:

```python

    """A simple greeting function."""
    message = "Hello, World!"
    print(message)
    return message

if __name__ == "__main__":
    hello_world()
```

```javascript

  console.log(`Hello, ${name}!`);
};

greeting('Reader');
```
